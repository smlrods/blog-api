# Blog API

## Getting started

```
git clone https://github.com/smlrods/blog-api.git
cd blog-api
npm install
npm start
```

## Built with

- [Express](https://expressjs.com/)
- [Passport](https://www.passportjs.org/)
- [Bcryptjs](https://github.com/dcodeIO/bcrypt.js)

## Database

- [MongoDB](https://www.mongodb.com/)
